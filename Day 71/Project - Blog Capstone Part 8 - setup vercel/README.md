
# Blogg  


Detta projekt är en bloggplattform byggd med Flask där användare kan skapa, läsa, redigera och ta bort blogginlägg. Plattformen använder en databas för att lagra inlägg, användare och kommentarer, och har säker autentisering för registrering och inloggning.

Applikationen är utvecklad med Flask som backend, SQLAlchemy för databashantering och Flask-Login för användarautentisering, medan formulär hanteras med Flask-WTF. Gränssnittet är designat med HTML och Bootstrap för en responsiv och användarvänlig upplevelse.

Plattformen inkluderar ett kommentarsystem där kommentarer kan tas bort av både inläggsförfattaren och kommentarskrivaren. Dessutom finns en kontaktsida som gör det möjligt för besökare att skicka meddelanden direkt till plattformens ägare via e-post.

Projektet är byggt för att demonstrera fullstack-webbutveckling med Python och Flask, med fokus på databashantering, autentisering, formulärhantering, e-postintegration och dynamisk rendering med Jinja2.
